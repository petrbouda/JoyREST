How to serialize and deserialize an entity

This article concerns how to transform a body from an incoming request to entity and how to entity transform and save in an outgoing response. We can do that using classes called Reader and Writer.

Reader and Writer have one method in common inherited from interface General. This method is called isGeneral and in this particular case it's a flag for whether Reader or Writer is dedicated for a concrete entity or for all entities of the given MediaType.

There are several available Readers and Writers in the package joyrest-utils apart from StringReaderWriter:

Transformer Name Description Default
StringReaderWriter Transform String Class to/from the request/response. YES
JsonReaderWriter Transform an entity to/from the request/response in JSON format. NO
JavaSerializationReaderWriter It is able to serialize java entities and send them over the net and then the same entities deserialize using the same classes. It is very fast but it is hard to keep the class compatibility. NO
HessianReaderWriter Implementation of hessian binary web service protocol. Official pages NO
