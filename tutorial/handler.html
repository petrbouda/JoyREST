How to integrate JoyRest application with Server

JoeRest application may use two ways of integration depends on the type of the server - HTTP Server or Servlet-based Server.

HTTP Server Integration

The main component ensured JoyRest <-> HTTP Server integration is called Handler. Every server has a specific way to integrate with JoyRest, every of them provides a different way to find out information about HTTP Requests and Responses. Requests and responses wrap server-specific requests and responses and provide some caching because of an optimization.

Handlers and server-specific request and response are included in individual jar files and these files are available as a maven dependency.
