Welcome to the JoyRest tutorial

This tutorial introduces all functionality provided by the REST Framework. Please, look at the GitHub examples from which are used some snippets shown below.

   JoyRest Examples
# Functionality Introduction Includes
1 Download How to download RELEASE dependencies from Maven Central repository and SNAPSHOTs from Sonatype.
2 Run Application How to use a configurer and an application config to create an application context and to run the application. Configurer, ApplicationContext
3 Server Handlers How to integrate JoyRest application with dedicate HTTP or Servlet-base Server. InternalRequest, InternalResponse
4 Application Context How to inject all needed objects which is used to build up an application.
5 First Route How to define routes using TypedControllerConfiguration. Route, TypedControllerConfiguration
6 Aspects How to implement new behaviour which intercepts requests and responses. Aspect
7 Exception Handling How to configure exception handlers to customize responses according to a type of the thrown exception. ExceptionHandler, TypedExceptionConfiguration
8 Custom Writer x Reader How to customize marshalling and unmarshalling. Writer, Reader
9 Joyrest Maven Plugin How to use Joyrest-Maven-Plugin to generate all needed information.
10 Integrate Server How to integrate still non-integrated HTTP server.
11 Integrate DI Framework. How to cover new dependency injection framework and use it to bootstrap the application.
